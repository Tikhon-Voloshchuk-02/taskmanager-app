# Task Manager API
Backend application for task management built with Spring Boot, MongoDB, and JWT authentication.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MongoDB
- Maven
- Docker

## Features
- User login with JWT token generation
- Stateless authentication with Spring Security
- Protected REST endpoints
- Task management API
- MongoDB integration

## Authentication

The application uses JWT-based authentication.

Example login request:

```http
POST /auth/login
Content-Type: application/json

{
  "username": "test",
  "password": "123456"
